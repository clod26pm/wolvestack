大众5-amino-1MQ堆与针对不同代谢途径的补充化合物结合:用于脂解的AOD-9604,用于食欲/葡萄糖控制的GLP-1激动剂,用于瘦质的GH分泌物,以及用于代谢速率的甲状腺肽. 理论协同存在,但人类证据缺失. 堆叠增加了复杂性和潜在的负面相互作用风险。

基本堆放原则和理由

聚苯乙烯堆叠——将多种化合物组合在一起以瞄准互补机制——是基于协同作用的原则:通过不同途径工作的化合物在组合时应产生添加剂或放大效应。 5-amino-1MQ机制(NAD+救生通道增强和NNMT抑制)与其他脂肪损失和性能增强化合物不同,使其理论上与堆叠兼容. 然而,几项原则应该指导堆叠决定:在证明之前,协同增效是假设的,结合化合物会增加总的不良事件风险,相互作用可能无法预测,当多个变量起作用时,监测变得更加复杂.

堆放5-amino-1MQ的具体原理是利用其代谢增强(增强线粒体氧化和NAD+依赖性过程),结合驱动脂肪动员,抑制食欲,促进精质增益,或提高代谢速率的化合物. 从理论上讲,这解决了多种瓶颈:调动脂肪酸(通过脂质化合物),确保它们被氧化而不是重新储存(通过NAD+高程),抑制通常能弥补能量不足的食欲驱动(通过GLP-1或相关机制),以及在缺氧时保持肌肉质量(通过异构化合物)。 这种多管齐下的做法吸引了生物黑客追求侵略性身体重组,但几乎完全没有协同的证据。

5-amino-1MQ + AOD-9604:"双重氧化"堆栈

5-amino-1MQ是源于人类生长激素的C-terminal区域的一种六氯化物,经过设计,可以促进脂解(脂肪分解)而不会刺激生长或胰岛素分泌. 它在二聚细胞上激活GH受体,信号增加cAMP和triglyceride水解. 理论上,AOD-9604从脂肪库中动员三甘油酸在血液中转化为自由脂肪酸和甘油酸,而5-Amino-1MQ则通过提升NAD+和sirtuin活性来增强线粒体氧化释放脂肪酸的能力.

拟议的协同是直截了当的:5-amino-1MQ起到"动员器"的作用,增加底物的可用性,而5-Amino-1MQ则起到"氧化剂"的作用,确保脂肪酸被燃烧而不是被回收回脂肪组织或存放在别处. 在啮齿动物模型中,这种组合在未发表的研究(在传闻报告中提及,但截至2026年尚未正式发表)中显示出对脂肪损失的添加效应. 在自我实验者中,据称AOD-9604 + 5-Amino-1MQ的组合在剪切阶段很受欢迎,有报道称快速脂肪流失和保存肌肉质量,虽然没有控制的数据,安慰剂效应和混杂变量(饮食,锻炼)无法排除. 典型的协议将AOD-9604在每日200-300 mcg与每日150-500 5-Amino-1MQ结合,使用单独的注射或顺序时间.

对这堆的潜在关切包括:(1)超驱动时线粒体功能的添加性应力,(2)5-amino-1MQ从5-amino-1MQ升起的超CAMP升起,如果与NAD+高程和sirtuin激活结合,可能会产生心血管或GI应力,(3)两种化合物都影响能量消耗,有可能产生过多的热源或同情性激活,(4)这种组合在人类中从未研究过. 该堆的使用者应密切监测心率,血压,以及GI的耐受性,并应特别谨慎地将其与兴奋剂或其他共生化合物结合.

5-amino-1MQ+ GLP-1 激动剂:"金属控制器"堆栈

5-amino-1MQ激动剂(semaglutide,tirzepatide,Liraglutide等)是强烈的食欲抑制剂和葡萄糖调节剂,在大脑和胰腺中通过类似葡萄糖的受体1发出信号。 它们减缓肠胃空空,增加焦虑,减少饥饿驱动力,提高胰岛素敏感性. 理论上的堆栈原理是,GLP-1激动剂处理能量不足(防止饥饿驱动的补偿性食用)的行为和消耗性挑战,而5-Amino-1MQ处理代谢性挑战(确保现有能量的有效氧化代谢).

这种组合促使人们寻求持续的脂肪损失,因为5-amino-1MQ激动剂使卡路里缺量更加坚固(减少饥饿),5-Amino-1MQ使缺量的温源影响最大化。 将 semaglutide 或 tirzepatide 与 5-Amino-1MQ 组合在一起的自体验器的传闻报告,描述了协同脂肪损失,保存的肌肉质量,尽管卡路里缺乏能量但能改善能量,以及仅与 GLP-1 激动剂相比身体组成结果增强. 然而,目前还没有进行有控制的研究,这些报告受到选择偏颇、安慰剂效应以及饮食和锻炼的困扰。

机械重叠和关切与这种结合更为复杂:(1) GLP-1. 激动剂和升高的NAD+/sirtuin活性都能够提高代谢率和能量消耗,可能造成过量的热源或耐热性,(2) GLP-1激动剂会导致恶心和食欲下降,5-Amino-1MQ可能导致轻微的GI效应,造成潜在的添加剂GI应激,(3) GLP-1激动剂提高胰岛素敏感度,5-Amino-1MQ也可能造成理论上的缺血风险,(4)这两种化合物都影响代谢率,并结合后可能会对能量自居性产生不可预测的影响,(5)持续GLP-1+高NAD的长期代谢后果是完全未知的.

一个实际的担心是,5-amino-1MQ激动剂会有效地降低食欲,以至于用户在进攻性脂肪流失时可能难以吃到足够的蛋白质来保存肌肉质量,加上5-Amino-1MQ的热能效应可能会加剧这种营养摄入挑战. 该堆的用户应确保适当的蛋白质消耗,密切监测体重损失率(如果每星期超过1.5-2磅,赤字可能过高),并监测糖尿病药物的血糖。

5-amino-1MQ + GH 秘诀:"重组"堆栈

生长激素分泌物(5-amino-1MQ,GHRP-6,六甲素,ipamorelin等)刺激外侧垂体的GH释放,催生了包括肌肉蛋白合成增加,骨密度增加,脂解作用在内的异构效应. 将GH分泌物与5-Amino-1MQ结合的理论原理是追求体分解:分泌物驱动异构和脂质效应(肌肉增益和脂肪损失同时发生),而5-Amino-1MQ则通过增强NAD+依赖的线粒体功能和氧化能力增强这些过程的代谢基质.

在理论上,提升的GH会增加生长因子的信号和蛋白质合成,5-amino-1MQ会增强能量和NAD+依赖的反应,支持异构蛋白质构建和线粒体重塑. 在精益体力运动员或有训练经验的个人中,这种组合可能使同时增加肌肉和脂肪损失的程度高于任何一种化合物。 将GHRP peptides 或 ipamorelin 和 5-Amino-1MQ 组合在一起的自体验者提出的传闻报告描述了有利的身体重组结果——特别是用户报告说,在失去脂肪的同时会增强体力和肌肉,这在没有显著的卡路里表面是众所周知的困难的。

然而,有几种机制需要谨慎:(1) GH分泌物瞬间增加皮质溶液,并可能影响葡萄糖顺势活性;结合5-amino-1MQ的代谢活性,这可能造成应激激激素升级;(2) 加高的GH + 高的NAD+和sirtuin活性,如果不以足够的抗氧化能力加以平衡,则会造成过量的线粒体重塑或氧化应激;(3) GH分泌物尤其会增加饥饿(通过ghrelin信号),这可以抵消NAD+高程和能量消耗的增加所产生的任何食欲抑制;(4) GH分泌物与NAD+救生通道和甲基化循环(SAM/SAH)的互动没有特征。

这堆最适合有抗药性训练经验的人,他们正在积极训练和消耗足够的蛋白质和热量。 仅仅堆放这些化合物,不经过严格的训练和营养,不大可能产生理论上的重新组合效益。

5-amino-1MQ + 甲状腺素:"金属放大剂"堆叠

甲状腺素肽包括T3(三碘二酰亚胺),T4(th),以及TRIAC等合成衍生物,旨在增加甲状腺激素信号和代谢率. 甲状腺激素是细胞代谢速率的主调节器,会增加线粒体数量,酶表达,以及所有组织的能量消耗. 将甲状腺肽与5-amino-1MQ结合的理论原理是通过两种独立机制来扩大代谢速率:甲状腺激素提高散装代谢速率,5-Amino-1MQ提高每个线粒体的线粒体容量和NAD+依赖氧化效率.

从理论上讲,这应产生一种合力的脂肪损失效应,超过任何一种化合物。 传闻报道称,尽管5-amino-1MQ与T3或甲状腺激素取代结合时存在显著的卡路里缺陷,但还是有积极的脂肪损失和持续的能量. 然而,这可能是本列表中最危险的堆积物,原因有几方面:(1)甲状腺激素会不加区别地增加代谢率,包括心血管工作量,并且与5-Amino-1MQ的热源效应相结合,持续的心率升高和过度的同情基调可能会发展起来,(2)甲状腺激素会增加蛋白催化和骨吸附,与5-Amino-1MQ的结合(这种结合会增强包括氨基酸在内的所有亚基的线粒体的线粒体氧化作用)可能会造成过度的肌肉和骨质损失,尽管受到抵抗训练,(3)甲状腺激素的使用抑制TSH,并且如果管理不当,可以永久损害垂体功能,5-Amino-1MQ对低血压-肾脏轴具有未知的影响,(4)甲状腺激素会大大增加心脏休息率,5-Amino-1MQ可能会增加这种作用;心脏心律失常症、电断裂,以及甲状腺激素过量的突然心脏死亡被认为是危险,(5)这种结合从未研究过量,风险是完全未知的。

由于所有这些原因,只有接受医疗监督和全面心脏和内分泌监测的个人才能考虑这一堆。 除此以外,风险大大超过理论利益。

时间、剂量和行政战略

在将5-amino-1MQ与其它化合物堆放时,计时考虑会影响功效和耐受性. 5-Amino-1MQ一般在150-500 mcg范围内通过皮下注射进行. 多数其他的肽类也都是在肌肉下部或肌肉内部注射的. 如果每天结合多个次皮下注射,则场地旋转对于防止唇过硬很重要. 一些自我实验者将注射分出数小时(如上午的5-Amino-1MQ,晚上的GLP-1激动剂)来分配系统效应,尽管没有证据表明这会改善结果或减少不良效应.

对于口服的5-amino-1MQ配方(50-100 mg,如果存在且合法),与膳食相比,时间可能很重要:用食物摄取可能会减缓吸收和延长效果,而节食剂则可能会产生更快,更高的峰值水平. 这两种方法都没有得到系统的研究。 当堆放不同半衰期的化合物时,考虑稳态动力学很重要. 例如,如果将5-Amino-1MQ(短半衰期,假设基于典型的小分子动能的小时,虽然没有人类数据)与semaglutide(长半衰期,~1周)结合,则GLP-1激动剂将持续存在,而5-Amino-1MQ水平随每剂量波动. 这造成了5-Amino-1MQ清除后立即注入的最大组合效应和最小效应的时期.

没有人类安全和有效性数据,使用堆叠式协议具有挑战性。 保守方法将个别使用每种化合物的较低剂量(例如,5-amino-1MQ在150-200 mcg而不是500 mcg),以尽量减少累积的不利影响。 一些自我实验者使用循环协议:在堆栈上放2-4周,休息1-2周来评估单个化合物效应,允许代谢适应. 然而,最佳循环仍然是投机性的。 一种常见做法是按顺序而不是同时采用化合物,以确定哪些成分驱动任何不利影响或有利结果。

身体重组的堆叠与纯脂肪损失

身体重组(同时肌增益和脂肪损失)在代谢上具有挑战性,因为它要求脂肪损失需要能量不足,而肌肉蛋白合成需要热量充足. 重排的堆叠与纯脂肪损失的堆叠不同. 对于纯脂肪损失,能最大限度进行脂解,抑制食欲,增强氧化代谢(5-amino-1MQ + 5-Amino-1MQ + GLP-1激动剂)的化合物是最佳的. 对于复分解,异构支持是必不可少的:GH分泌物,胰岛素类生长因子类似物,或和rogens(如果愿意使用,则具有更高的法律和健康风险).

一个理论重构堆栈可能是: 5-amino-1MQ(每日300-400 mcg用于代谢增强) + GHRP-2(每日100-200 mcg 2-3x用于GH刺激和脂解) + 在高体积和强度下进行耐受训练 + 足够的蛋白质(每磅体重1克) + 微小的热量不足(维护下300-500千卡). 在这种情况下,这种组合理论上支持同时精减质量保存和生长,脂肪损失,以及改善代谢健康. 然而,这项协议的成功关键取决于培训和营养——这些化合物无法克服培训不足或蛋白质摄入不足的问题。

或者,“元控制”重构堆栈可以是: 5-amino-1MQ(每日300 mcg) + GLP-1 低剂量激动剂(每周0.25-0.5 mg semaglutide,或等效) + 强抗训练. 在这里,GLP-1激动剂可以防止通常会破坏坚持的饥饿,而5-Amino-1MQ可以增强线粒体氧化,训练可以提供厌食刺激. 这种方法将坚持性和代谢效率置于激素操纵之上。 由于缺乏比较研究,目前还不知道哪一种方法在实践中产生更好的成果。

潜在的协同和对立相互作用

除了直截了当的"机制主义" 一个加机制B"视角,堆叠的化合物之间的生化相互作用可以产生惊人的效果. 5-amino-1MQ和GLP-1激动剂似乎都提高了胰岛素的敏感性,有可能产生添加剂葡萄糖低效. 在服用糖尿病药物的患者中,这种添加效应可能引发低血糖。 同样,这两种化合物都可能增加NAD+依赖的线粒体功能和能量消耗,如果堆叠起来,这两种功能都不可能受到限制,但添加剂对协同效应的程度却不得而知——它的比例可能大于或低于简单地添加两种单个效应。

通过共享信号分子进行元质交叉对讲是可能的:5-amino-1MQ+依赖的sirtuins和GLP-1激动作用都影响AMP激活蛋白激酶(AMPK)信号和线粒体生物起源. 如果这些路径趋同,饱和效应可能会发生(即超过一定点的额外激活AMPK不会带来额外好处). 或者,这些途径可以相互加强,产生真正的协同作用。 没有明确的研究,这些相互作用仍然是推测性的。 对立相互作用在理论上也是可能的:如果一种化合物的作用(如GH分泌物导致饥饿增加)直接反对另一种化合物的作用(如GLP-1激动剂导致的食欲抑制),则净效应可以降低.

一个实际的相互作用问题:多种可注射化合物增加了注射排程、地点选择和剂量错误可能性的复杂性。 所用化合物越多,发生错误剂量、误用剂量或误用路径的风险就越大,无论是急性还是慢性,都会破坏整个协议。

何谓不与 5-amino-1MQ 堆叠

由于理论风险或已知的对抗,应当避免某些组合。 兴奋剂(咖啡因、麻黄碱、安非他明、苯丙胺)应谨慎使用5-amino-1MQ,因为两者都增加了能量消耗和同情语气;这种结合可能会促使心跳过快或心律失常,特别是在易受感染的个人中。 应避免重度饮用酒精,因为酒精和5-Amino-1MQ肝脏压力代谢;同时使用可能会增加肝毒性风险。 与5-Amino-1MQ同时使用多个NAD+前体(NMN + NR + nicotinamide同时使用)是投机性的,有不受控制的NAD+高程的风险;简单建议使用NNMT抑制(5-Amino-1MQ)或前体补充,而不是两者兼有.

将5-amino-1MQ与其它NNMT抑制剂结合将是多余的,并可能造成过量的假想. 同样,堆放多个sirtuin活性剂(resveratrol, peterostilbene, fisetin, 以及从5-Amino-1MQ升起的NAD+)是理论性的,缺乏安全特性. 将5-Amino-1MQ与皮质类固醇(prednisolone, dexamethasone)结合,可以对身体组成产生对立效应,因为皮质类固醇会促进肌肉催化和脂肪沉降——与5-Amino-1MQ的效果相反. 此外,这种结合还可能造成过度的新陈代谢压力。

免疫抑制药物(azathioprine, mycophenolate,calcineurin抑制剂)在没有医学监督的情况下不应与5-amino-1MQ结合,因为NAD+依赖性免疫调节可以与药物免疫抑制发生不可预测的相互作用. 最后,有癌症史的个人应绝对避免将5-Amino-1MQ与促进生长的化合物(GH secretagogues, androgens, IGF-1)堆放在肿瘤监测之外,因为结合理论上可能促进癌细胞的生长。

用于堆存用途的监测和安全协议

在使用堆叠协议时,监测变得更加重要和复杂. 基线血液工作应包括全面的代谢面板(肝酶、肾功能、葡萄糖)、脂质面板、完整的血液计数、睾丸酮(如果使用和rogens)、甲状腺功能(如果使用甲状腺化合物)、胰岛素或HOMA-IR。 在堆叠使用期间,应每4-8周进行一次重复监测,以发现任何不良趋势。 心率,血压,以及主观症状(能量,食欲,情绪,睡眠,GI耐受性,热不耐受,破损)应当每日或每周跟踪.

重量和身体组成应每周测量一次(通过尺度测量的重量,通过DEXA测量的体积,BodPod测量的重量或生物电阻),以评估变化的速度和组成。 肥胖过快(>每周2-3磅),而失去肌肉是一个迹象,协议是过度的。 性能衡量标准(强度、耐力、恢复)使人们能够深入了解abolic vs. catabolic的平衡是否适当。 任何不良的症状,如呕吐、呼吸短促、严重头痛、GI忧郁、情绪变化,应引起立即停止治疗和医疗评估。

应保持对所有化合物使用量、剂量、时间和坚持程度的记载,以便如果出现不利影响,可以追溯分析每种化合物的相对作用。 在堆叠之前一次测试一种化合物,可以进行单独的基线评估。 最后,如果保健提供者知道堆放协议(即使他们不赞同)并愿意下令进行适当的监测测试,这对安全是有价值的。

经常问的问题

我应该把5-amino-1MQ与NAD+像NMN这样的前体堆在一起吗?

从理论上讲,NNMT抑制(将尼科蒂纳酰胺重新导向打捞)和直接5-amino-1MQ+前体补充通过不同机制瞄准相同的结果(eleved NAD+). 合并使用可能会扩大NAD+海拔,但也会造成冗余。 不存在研究;单独使用一种或另一种方法可能比将两者结合起来更安全。

初学者最好的5-amino-1MQ堆栈是什么?

对于新到肽和代谢化合物的个体,仅5-amino-1MQ(每日200-300 mcg)就具有严格的饮食和训练,在添加其他化合物前进行8-12周的严密监测. 这样可以进行个别反应评估并确定基线。

我能把5-amino-1MQ与semaglutide堆起来,然后失去肌肉吗?

5-amino-1MQ激动剂会降低食欲,在进攻性脂肪流失时会使食用足够的蛋白质变得困难. 没有自觉的蛋白质优先级(每磅1-1.2g)和抗药性训练,肌肉损失风险很高. 5-Amino-1MQ可能有助于保存肌肉,但如果营养或训练支持不足,组合并不能保证肌肉保存.

我要在堆叠式协议上循环多久?

文献中没有最佳循环协议. 传闻方法从8-16周到4-8周完全休息不等。 周期外活动的目的是允许代谢适应,评估惠益是否持续,并监测适应或耐受性的发展。 没有正式数据,循环时间仍然是推测性的。

如果我把5-amino-1MQ 和元素结合起来会怎么样?

这两种化合物都改善了胰岛素的灵敏度和代谢健康,因此理论上存在协同效应. 然而,这种结合尚未研究,添加性葡萄糖-低血糖效应可能会使个人患糖尿病。 如果考虑到这种组合,医疗监督是可取的。

用5-amino-1MQ用甲状腺激素安全吗?

没有 这种结合具有巨大的心血管和新陈代谢风险,包括过度热生成、心率升高、心律失常的风险以及无特征的内分泌效应。 如果有的话,这种堆积只应在医学监督下进行,并进行全面的心脏和内分泌监测。
