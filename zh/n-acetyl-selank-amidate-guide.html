了解Selank及其化学增强

Selank是俄罗斯研究者于1970年代开发的一种合成肽,是一种麻醉剂. 原始的Selank肽(序列:Thr-Lys-Pro-Arg-Pro-Gly-Pro)是一种七聚苯二甲酸酯,由tuftsin衍生而来,是一种内源性免疫代谢肽. Selank虽然有效,但有局限性:酶迅速降解,血液脑屏障(BBB)渗透不良,半衰期短,需要频繁服用。

Selank(NAc-Selank-酰胺)通过两个关键修改来解决这些限制:

  1. N- 标记 : 将乙酰基团添加到N-terminus(自由氨基团)中,可以保护肽不受氨基乙酰基团的快速酶降解. 这延长了半衰期,提高了代谢稳定性.
  2. 更改对象 : 将C-terminal carboxyl组转化为酰胺可改善脂溶性(脂肪溶性),促进更好的血脑屏障渗透和细胞吸收。

Axiolytic 机制: GABA 调制和 Selank

Selank通过多个互联机制减少焦虑:

GABAERGIC 路径 : GABA(γ-氨基丁酸)是大脑的主要抑制神经递质. 焦虑的部分原因是与抑制性语气(GABA)相比,过度的排泄性语气(glumatate administration). NAc-Selank-酰胺通过增强GABA合成和受体敏感度,增强GABAergic信号,将激发-抑制平衡转向平静.

与苯二氮杂卓(直接绑定GABA-A受体)不同的是,NAc-Selank-酰胺通过更微妙的机制调节GABA——将谷氨酸脱羧酶(GAD),即从谷氨酸合成GABA的酶,并有可能增强GABA受体的表达. 这导致在不引起苯并二氮杂卓的镇静、容忍或依赖性责任的情况下减少焦虑。

大脑神经营养因子(Selank): Selank是神经弹性、学习和情绪调节所必需的生长因素。 慢性压力和焦虑与BDNF下降有关,特别是在对记忆和情绪调节至关重要的河马营和前额皮层地区。 NAc-Selank-酰胺可以调节BDNF表达,促进神经起源和抗压能力.

单胺调制: 一些研究表明,NAc-Selank-酰胺通过对再摄取或合成的影响,适度地增加了多巴胺和血清素的可得性. 这有助于情绪升降和动机,而不需要SSRI抗抑郁药的强度.

HPA轴线条例: 下丘-医院-肾上腺(HPA)轴通过皮质溶液释放协调应激反应. 慢性焦虑与HPA多活性有关. NAc-Selank-酰胺温和HPA基调,在不完全压制轴时降低皮质溶液的阻力(这会导致不适应性). 压力激素的这种正常化有助于血管解析.

俄罗斯临床研究:证据基金会

Selank产生于俄罗斯的药物研究,研究范围远大于西方出版物. 俄罗斯临床试验(主要在1990-2010年代进行)构成主要证据基础:

减少焦虑研究: 俄罗斯对普遍焦虑障碍患者(GAD)的多次随机控制试验显示,NAc-Selank-酰胺(每日两次为100-200 mcg内脏)比安慰剂显著降低焦虑症状. 症状在3-7天内出现改善,并在4-8周内继续改善。 反应率(定义为焦虑分数降低50%或更高)约为60-75%.

与苯二氮杂卓的比较: 在对标准麻醉药(diazepam,phenazepam)进行头对头试验时,NAc-Selank-酰胺在2-4周内产生类似的焦虑降低. 关键是,NAc-Selank-酰胺没有引起镇静剂、认知障碍或耐受性——限制苯并二氮杂卓长期使用的副作用。

压力引起的焦虑: 与安慰剂相比,对接触急性心理压力(身体测试、公开演讲情景)的健康志愿者的研究显示,NAc-Selank-胺预处理减轻了焦虑和生理压力标记(cortisol,心率)。 低至100 mcg的剂量显示出可衡量的效益。

睡眠和焦虑社区: 俄罗斯研究指出,NAc-Selank-酰胺常改善焦虑病人的睡眠质量,不是通过镇静剂(如苯并二氮杂卓),而是通过减少焦虑,允许自然睡眠. 一些研究具体测量了这一点,发现尽管夜间睡眠情况有所改善,但白天没有镇静剂。

研究背景

为什么是俄罗斯Data? Selank是在俄罗斯开发并广泛研究的,俄罗斯已经批准其作为麻醉剂临床使用,品牌名称为Selank-Amidate. 西方出版的文献很少, 然而,俄罗斯的临床数据库是实质性的,在方法上受到国际标准的严格。

多辛协议:内部管理

Selank的标准剂量是鼻内(鼻部喷雾或粉末浸润),它能提供快速的BBB访问,避免肝先通过代谢.

标准协议 :

  • 剂量 : 每个行政部门200-600 Selank
  • 频率 : 每天一到两次(上午和持续焦虑的可选晚上)
  • 路线 : 内部喷洒(常见于250 Selank/喷瓶)或粉末浸渍
  • 行政部门: 直立地坐在一个鼻孔里 轻轻的嗅觉以方便鼻腔分布. 不要马上吹鼻涕(吸收时间2-3分钟)。
  • 设定 : 15-30分钟
  • 持续时间 : 每剂4-8小时

装入和维护: 与需要蓄积的肽不同,NAc-Selank-酰胺作用于第一剂量. 然而,随着神经适应(BDNF 上升调节,GABAergic语调)的发展,焦虑减少往往随着连续7-14天的日常剂量而得到改善.

弹性剂量 : 许多用户采用"视需要"的方法——在焦虑症发作时或在预期的紧张情况之前,使用300-600 Selank. 其他人使用维持剂量(每日200-300 mcg)来保持基线平静. 这两种方法都是有效的。

无镇静剂抗焦虑:关键区别

Selank的关键优势是它的无镇静剂机制,它比传统安乐剂更强. 理解为什么重要:

苯二氮杂卓: 这些药物直接将GABA-A受体结合,增强整个大脑的抑制语气. 这既会减少焦虑,也会损害警惕性、运动协调以及认知效应,许多用户认为这不可取。

纳克-Selank-Amide: 这种肽通过GABA增强(但不直接受体绑定),Selank提高调节,以及单胺效应来调节焦虑. 这些机制优先影响情绪处理电路(amygdala,前额皮质),没有全球CNS抑郁症. 其结果是:不发生镇静、认知障碍或运动效应而减少焦虑。

俄罗斯临床数据明确测量了NAc-Selank-酰胺实验对象的警惕性,反应时间,认知功能. 与损害这些参数的苯二氮杂卓不同,NAc-Selank-酰胺使警惕性保持不变或稍有改进(通过BDNF对认知的影响).

不是替补

虽然Selank具有麻醉性,但并不能完全替代严重焦虑症或恐慌症治疗. 中度焦虑和情势焦虑反应最好. 对于严重,耐治疗的焦虑,可能需要常规药物.

Selank 和 长期中转

NAc-Selank-酰胺最有趣的方面之一是它的BDNF-促进作用,这可能有助于除肽的直接药理作用外的长期焦虑韧性.

Selank在焦虑中的角色: 脑衍生神经营养因子支持情绪调节电路中的神经弹性和突触稳定性. 慢性应力降压Selank,损害突触可塑性和情绪调节. 反之,BDNF高地可促进对威胁探测电路的适应性改造,并加强对amygdala反应的预先控制。

持续效果 : 一些俄罗斯的研究表明,即使在终止NAc-Selank-酰胺后,焦虑的减少持续时间也比eptide2-4小时半衰期的预期要长. 这归因于BDNF驱动的神经塑性变化,这些变化在肽的直接存在之外持续存在. 如果复制,这意味着NAc-Selank-酰胺可以产生持久的好处,而不仅仅是症状缓解。

与认知行为治疗的叠加: 理论上,NAc-Selank-酰胺的BDNF高程可以通过在治疗学习中优化神经弹性来提高CBT的功效. 俄罗斯的一些临床医生建议将NAc-Selank-酰胺与结构化的焦虑缓解疗法相结合,以产生协同效应. 然而,这在西方背景下仍然是理论性的.

副作用和安全简介

Selank根据临床数据,有特殊的安全简介.

不利影响(按频率):

  • 常见 : 无(副作用率最低)

相互作用: 没有关于毒品与毒品之间严重相互作用的记录。 NAc-Selank-酰胺可以与大多数精神药物(SSRIs,SNRIIs,抗精神病药物)结合而无并发症,尽管与苯二氮杂卓或酒精结合是不推荐的(协作效应未知).

经常问的问题

Selank工作有多快?
内部剂量在15至30分钟内会产生明显的麻醉作用. 发病速度比SSRI(需要几周)快,但比苯并二氮杂卓慢(5-15分钟). 随着Selank和GABAergic适应的加深,充分的好处往往在7至14天的时间内得到发展。
我可以按需要使用它 或者我必须每天注射?
这两种方法都有效。 一些用户每天用一次或两次剂量作为维护(200 Selank晨,可选200 mcg晚). 其他人在焦虑症发作(300-600 mcg)时根据需要使用. 由于BDNF的积累,每日持续剂量可能会产生更好的效果,但根据需要剂量对急性焦虑情况有效.
它是否会引起镇静或损害认知功能?
没有 与苯二氮杂卓或抗西胺不同,N-Acetyl Selank Amidate不会引起镇静剂或认知障碍. 临床数据显示警戒性没有变化或略有改善. 由于BDNF对内存和学习的影响,一些用户报告了轻微的认知增强.
我能把它和苯二氮杂卓或抗抑郁药结合起来吗?
与SSRI或SNRII的结合是安全的——两者都是通过不同机制进行的目标焦虑,没有记录的不良相互作用. 在没有医疗监督的情况下,不建议与苯二氮杂卓结合(协作效应未知,尽管NAc-Selank-酰胺是非镇静剂,但过量镇静剂的风险)。 避免与酒精结合。
是否容忍或依赖长期使用?
没有 俄罗斯的临床数据从数月到数年的使用都表明没有耐受性(丧失效力)或依赖性。 用户可以随时中止,而无需出现戒断症状. 这比苯并二氮杂卓有较大的优势.
和SSR相比,
SSRIs(Sertraline,paroxetine等)需要4-6周才能工作,目标情绪广泛,通常导致性功能障碍和体重增高. NAc-Selank-酰胺在15-30分钟内工作,具体针对焦虑,不产生性或代谢效应,不承担依赖责任. 然而,SSRI对于中度严重焦虑症可能比较优越. NAc-Selank-酰胺在轻度中度焦虑和情况性使用方面表现优异。
