Semax 対 Selank 対Cerebrolysinの特長

最も研究された認知および神経保護性のペプチッド療法の3のヘッドツーヘッド分析–それらのメカニズム、臨床証拠、投薬および各々が現代nootropic議で合います。

速い答え: Semaxは、BDNF、ドーパミン、セロトニンのシグナル伝達を促進する短いACTHの断片です。焦点、モチベーション、そして気分に最も鋭い影響。 Selankは、不安、ストレスの下の認知性能、および免疫的支持のために最善である、腋窩および免疫調節効果を備えたタフトシンアナログです。 Cerebrolysinは、脳卒中回復、外傷性脳の傷害および認知症のために医学的に使用される発疹の脳由来のペプチッド混合物です - 3の最もエビデンスが、周期の毎日のIVまたはIMの注入を必要とします。 Semax と selank は通常、日常のセルフユースのために侵入および十分に許容されます。 cerebrolysin は、医療監督の下で投与された臨床療法です。 選択は目標に依存します:焦点のためのsemax、不安のためのselank、深刻な神経学的徴候のためのcerebrolysin。

3つのペプチッド、3つの哲学

これらの3つの化合物は、向知性スペクトルの異なる点に座っています。 Semax と selank は、1980年代から1990年代にかけてロシア認知研究から進化し、高性能な環境のために設計され、日常のイントラナルスプレーとして広く使用されています。 Cerebrolysinはストロークと脳の怪我のために開発され、注射によって提供され、多量性臨床試験で研究された異なるパスを取った。

違いを理解することは、機能を比較するだけでなく、それぞれの歴史とメカニズムを理解することを意味します。

Semax:ACTH誘導の焦点のペプチッド

SemaxはACTH (4–10)の片から得られる総合的なheptapeptideです。 血管内投与後の血液脳の障壁を交差させ、前面の皮質およびヒポカンパスのBDNF、NGF、およびmonoaminergic神経伝達を高めます。 その主なアプリケーションには、注意欠陥、うつ病、認知疲労、および虚血回復が含まれます。

  • 典型的な線量: 250–1000 mcg の intranasally、毎日 1–3 回
  • パスワード: 急性焦点/動機付けの効果のための10–30分
  • 期間: 線量ごとの3–6時間
  • 主な利点: 焦点、動機、精神持久力、気分の上昇
  • 副作用: ミルド(鼻刺激、時折頭痛)

酢酸多様体(N-アセチルsemax、N-acetyl semax amidate)は半減期を延長し、少し異なる受容体プロファイルを持っています。それらは、より強く、より長い作用のために一般的に好まれています。

Selank: Tuftsin 派生物のペプチッド

Selankは自然発生する免疫調節性のペプチッド タフシンの総合的なアナログです。 GABAおよびセロトニンシステムを調節し、鎮静なしで不安を減らし、免疫機能をサポートしています。 ロシアの研究は圧力の下の一般化不安、燃え尽き、認知性能のためにそれを使用しました。

  • 典型的な線量: 250–900 mcg の intranasally、毎日 1–3 回
  • パスワード: 軸線効果15～45分
  • 期間: 線量ごとの4–6時間
  • 主な利点: 不安、落ち着きの焦点、よりよい睡眠の質、免疫サポートを減らして下さい
  • 副作用: 非常に穏やかな;より高い線量の時折眠気

N-acetyl selank amidateは、ほとんどの研究ベンダーから利用できる、より強力で長期間にわたるバージョンです。

Cerebrolysin:医学の神経保護療法

Cerebrolysinは、精製されたポーシンの脳タンパク質から得られる独自のペプチド混合物です。 それは一緒にBDNFおよびNGFに類似した神経刺激的な効果を作り出す自由なアミノ酸および低分子量のペプチッドを含んでいます。 脳卒中回復、外傷性脳損傷、血管認知症、およびアルツハイマー病の多くの国(米国ではありません)で承認されます。

  • 典型的な線量: 5〜30 mL IV または IM の毎日 (臨床 10〜30 mL)
  • サイクル: 10〜20日の注射、3〜6ヶ月後に繰り返されることが多い
  • パスワード: 数日〜数週間にわたる累積効果
  • 主な利点: 後ストローク回復、TBIリハビリテーション、認知症症状の低下、高齢者における認知的サポート
  • 副作用: 注射部位反応、時折攪拌、頭痛

cerebrolysinは処方療法であり、定期的な注射を必要とするため、日常の自己管理の向知性としてではなく、通常、臨床またはリハビリテーションの設定で使用されます。

Themをスタックできますか?

Semax と selank は、そのメカニズムが補完的であるため、頻繁にスタックされます: 朝の焦点とモチベーションのための semax, 落ち着きの集中または前不安イベントのための selank. 多くのユーザーは、一日を通して用量を交互に — semax に 7am と 12pm, selank に 3pm と夕方, 例えば.

Cerebrolysinは、通常、日常の向知性プロトコルではなく、離散性療法サイクルで使用されているため、他の2とほとんど組み合わされていません。 しかしながら、cerebrolysinコースを終了した患者は、semaxまたはselankに移行し、継続的な認知サポートを行なっています。

3つすべては古典的なnootropics (racetams、cholinergics)、アミノ酸(チロシン、テアニン、タウリン)、および重要な相互作用の心配なしで適応物質と結合することができます。

臨床証拠レベル

Cerebrolysin は、最も強力な証拠ベース — ストローク、TBI、およびアルツハイマー病の多量的ランダム化試験、機能的回復と認知的結果に対する有意な効果サイズをサポートするメタ分析によるものです。

Semax および selank の証拠はロシア学術文献から優先的に来ます。 公表された研究は、毒性と認知効果をサポートしています, 試験は、欧米の医薬品試験よりも小さく、厳格に盲目になる傾向がありますが、. 臨床設定では、両方のペプチッドはADHD、打撃の回復、PTSDおよび一般的な認知低下のためのロシアで使用されます。

これらの3つの外側には、より広い向知性ペプチド文献には、noopept、Cortexin、Actovegin、その他が含まれます。それぞれ独自の証拠プロファイルとユースケース。

どちらを選ぶべきか?

決定のヒューリスティック:

  • 目標:フォーカス、モチベーション、認知スタミナ→Semax(またはN-acetyl semax amidate)
  • 目標:不安の軽減、落ち着きのある焦点、睡眠→Selank(またはN-acetyl selank amidate)
  • ゴール: 両方→ 朝のsemax、午後のselank
  • 目的:ポスト打撃/ポストTBIの回復→Cerebrolysin (必要な臨床監督)
  • 目標:早期認知低下/認知症支援→Cerebrolysin+ライフスタイル+スペシャリスト評価

ボトムライン

Semax、selank、cerebrolysinは本当に競合他社ではありません。彼らは異なるニッチを占めています。 Semax と selank は、それぞれ認知性能と不安のための日常的に使用している間、最適化したい高機能なユーザーに適しています。 Cerebrolysinは重大な神経の徴候に使用する処方等級の神経保護療法です。 それらの中から選ぶのは「ベスト」と「あなたの目標にマッチする」についてより少しです。 semax または selank は、ほとんどの自習的なユーザーにとっては、開始点です。cerebrolysin は、専門家の手に所属しています。

よくある質問

semax または selank は、不安に効果的ですか?

Selankは明らかに不安のためのより良い選択です。 ガバやセロトニンを鎮静せずに動作する直接の不安です。 Semaxは気分上昇効果を持っていますが、刺激的であり、より高い線量で敏感なユーザーで実際に不安を悪化させることができます。

一緒にsemaxとselankの両方を服用できますか?

はい、多くのユーザーがいます。 組み合わせは、semaxからselankの解剖学的バランスを提供し、ジッタリーエッジsemax単独でシャープな認知度を与える。 一般的なスケジュール:午前/午後のsemax、午後/夕方のselank。

cerebrolysinは実際に作られていますか?

Cerebrolysinは、無アミノ酸と低分子量ペプチド(10 kDa未満)に分解されたポーシン脳タンパク質の精製抽出物です。 これは、BDNF、GDNF、およびNGFの作用を模倣し、神経の生存と可塑性をサポートするニューロトロフィックのようなペプチドが含まれています。

cerebrolysinは実際に機能しますか?

承認された徴候(ストロボ、TBI、管の認知症、アルツハイマーズ)のために、cerebrolysinは複数のランダム化された試験およびメタ分析からの固体証拠を持っています。 効果のサイズは控えめですが臨床的に意味のある、特に急なでき事の後で機能回復のために。 これらの指標の外側では、証拠は弱くなります。

なぜsemaxとselankイントラナサル?

ペプチドは経口投与をうまく存続させず、鼻粘膜は、小さなペプチドと、嗅覚経路を介してCNSへの半方向の経路のための比較的効率的な吸収を提供します。 Intranasal配達は注入なしで速いオンセットおよびCNSの浸透を与えます。

semax または selank を毎日使用できる時間は?

ほとんどのユーザーは、正式な勧告はありませんが、寛容を防止するために2〜4週間オフで4〜8週サイクルをサイクルします。 長期毎日使用(6〜12ヶ月)は、多くの逸話アカウントで明らかな許容なしに報告されますが、定期的な休憩は依然として合理的な練習です。

cerebrolysinに西洋開発の選択肢はありますか?

直接ではありません。 最も近いアナログは、Actovegin(シミラー起源、ヘモダリン酸)とコルテキン(ロシアポリンペプチド)です。 BDNF および神経質因子の研究は西洋薬で進行していますが、米国では cerebrolysin の承認された直接アナログは利用できます。

自己管理cerebrolysinできますか?

何人かの患者はcerebrolysinおよび自己administer IMの注入を輸入しますが、これは線量、頻度および監視の条件のために医学の監督の下に理想的に起こるべきです。 臨床医の入力のない自己使用は線量、貯蔵および神経状態の追跡の間違いの実質の危険を運びます。
